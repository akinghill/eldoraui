## Props

| Prop Name   | Type     | Default | Description                                                       |
| ----------- | -------- | ------- | ----------------------------------------------------------------- |
| `color`     | `string` | `#0F0`  | The color of the characters in the background (default is green). |
| `fontSize`  | `number` | `14`    | The font size of the characters in pixels (default is 14).        |
| `className` | `string` | `""`    | An optional custom class name for styling.                        |
| `speed`     | `number` | `1`     | The speed at which the characters fall (default is 1).            |
