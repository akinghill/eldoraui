## Props

| Prop Name   | Type     | Default | Description                                                          |
| ----------- | -------- | ------- | -------------------------------------------------------------------- |
| `textLeft`  | `string` | `""`    | The text content to display on the left side of the slide.           |
| `textRight` | `string` | `""`    | The text content to display on the right side of the slide.          |
